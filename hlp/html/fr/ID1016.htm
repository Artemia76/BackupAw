AW

The AW Menu provide all tools to manage the AW connection.

Connect/Diconnect

Connect the bot on AW world.

 

Setup



a dialog box to set the few parameters needed to establish an AW connection


Citizen : Enter the citizen number for priviledge using

Password : Enter the priviliedge password of the citizen

Universe : The IP or host of universe target

Port: The port number of the universe.

Bot Name : The name of the bot on world

(for backupAW not usefull because it stay inivisble)

World : The world name to connect the bot.

Reconnection Delay: Time to wait before attempting a new reconnect after

a lost univers or world conenction.

Number of attempt: 0 for never, 1 to 9 for number of attempt, -1 for infinite.

Auto connection on startup : Enable the automatic connection on startup.



Note : The .BAP file format is editable with a notepad for handing operation

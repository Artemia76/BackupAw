OVERVIEW

Backup Aw is a survey backup tool for ActiveWorlds technology.

Its main function is to help the caretaker for his admin tasks :

- Backup and Restore local builds
- Manage Citizen terrains
- Clean the tourist area
- Move or Rotate
- Give a colored ( I.R like) density of objects on world.

It doesn't make the coffee yet ;-)

My personal convictions to realize this bot is : more easier posible
and intuitive use.

 Help for Main Menu of BackupAW

Mouse Operations

Left mouse button: make a region selection automatically reported to the filter tab.

Middle and Right Mouse button : Drag the grid in another center coord point.


Density and Cell

The Grid represent the cell in satellite view of the current world.

After Scan operation, you can see the cell density in IR (Infra Red ) Colours

White : The cell is empty

Dark blue : The cell contains 1 object

Red : The cell contains near 10 objects

Yellow : The cell contains more 20 objects

Each grey cells have 10 meters square.

Each blue cells have 80 meters square and are called Sectors

Filter Tab:

The Filter tab provide a powerfull tool to select only the object you want

you just have to set the filter and press UPDATE button to activate the project selection

Citizen



Enable : Enable the citizen filter on global project

Exclude : Exclude the specified citizen number and select all others if enable
else, only citizen owned objects are selected.


Model

Enable : Enable the model name filter on global project

Exclude : Exclude the specified object name and select all others if enable
else, only named objects are selected.


Description

Enable : Enable the Description content filter on global project

Exclude : Exclude the specified the objects who contain the Description string if enable
else, only the objects with the Description String will be retained.

Exact Match : Retain only the exact form if enable else, a part of the Object Description
will be enought

Action

Enable : Enable the Action content filter on global project

Exclude : Exclude the specified the objects who contain the Action command if enable
else, only the objects with the Action Command will be retained.

Exact Match : Retain only the exact form if enable else, a part of the Object Action
will be enought

Region

Enable : Enable the Region filter on global project

Exclude : Exclude the objects inside the region Area if enable
else, only the objects inside the region will be retained.

NW Corner : The top left corner of the desired Region

SE Corner : The buttom right corner of the desired Region

 


Citizen Modifier



In List , You can see the list of objects ranged by citizen number in project

you can change a group of x citizen number to an y. To do that enter in

the left case the source citizen number (Use -1 value to mean ALL)

and in the right, the destination citizen number

Then ,click on Change. The operation take effect only in the project (not in the world)

Move



The "Move" tool provide you the possibility to move complete build in 3D.

From and To represent a translation vector for movment.

the coordinate can be entered in AW format with decimal point for more definition

When ready to move, just click Move. The Move take only effect in the project.

NOTE : The lowest unit on AW is the centimeter : 0.01 N for example.

Rotate



The "Rotate" tool modifier give you the possibility to rotate entire build around a pivot point and
a counter clock wise relative angle in degrees.

Keep in mind that the coordinate system of AW can't go under the centimeter. Backup aw compute all
in float value.So the result under AW can make hole between object (very small) if you are use
not quadrant angles (90 180 270).

Pivot Point is the pivot coordinate point in AW format.

CCW Angle is the Relative Angle for the rotation.

When ready, click on Rotate button to apply rotation in project.

Build Tab

The "Build Tab" is used to build an active project on memory on the world.

Check the Build As a CareTaker if you need to have full world building options.

IMPORTANT : The BackupAw build over the existant object, so clean
the destination terrain before build on it.

 

Delete Tab

The "Delete" is used to destruct an active project.

CAUTION : BackupAw use the coord and ID of each object to make reconize them
by AW world Server : So don't use Modifiers before a delete opeation

PROJECT

The Poject menu manage the files operations and exiting of the software.

Load


Load a project stored in a .bap file format.


Save

 

Save the current project in memory to a .bap file format


Save_As


Save the current project under another name in a .bap file format


Exit

Exit BackupAw

Note : The .BAP file format is editable with a notepad for handing operation.
